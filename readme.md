# Arabic Keyboard Translator

Fix text typed with the wrong keyboard layout.

Examples:
- hojfhv → اختبار
- فثسف → test

## Features
- English → Arabic keyboard conversion
- Arabic → English keyboard conversion
- Works directly in the browser
- No installation required

## How to use
1. Open the website
2. Paste text typed with the wrong keyboard layout
3. Click the conversion button
